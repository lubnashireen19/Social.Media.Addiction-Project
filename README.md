# 📱 Social Media Addiction Prediction (Machine Learning)

## 📌 Project Overview

This project uses Machine Learning to predict social media addiction levels among students based on behavioral and usage patterns.

---

## 📊 Dataset

* 705 rows and 13 columns
* Features include:

  * Age
  * Gender
  * Academic Level
  * Country
  * Daily Usage Hours
  * Sleep Hours
* 🎯 Target Variable: **Addicted_Score**

---

## ⚙️ Steps Performed

* Data preprocessing
* Handling missing values
* Label encoding (categorical variables)
* Feature extraction
* Feature scaling
* Train-test split
* Linear Regression model

---

## 📈 Results

* Mean Squared Error (MSE): **0.107**
* R² Score: **0.957**

✔ The model shows excellent performance in predicting addiction levels.
